# Claude AI Development Guidelines / Claude AI 開発ガイドライン

## Project Overview / プロジェクト概要

This is a template repository for starting new projects with AI-driven development support.

これはAI駆動開発サポート付きの新規プロジェクトを開始するためのテンプレートリポジトリです。

## Development Principles / 開発原則

### Code Quality / コード品質

- Write clean, maintainable, and well-documented code
- Follow established coding conventions and style guides
- Prioritize readability and simplicity
- クリーンで保守性が高く、十分に文書化されたコードを書く
- 確立されたコーディング規約とスタイルガイドに従う
- 可読性とシンプルさを優先する

### Testing / テスト

- Write comprehensive unit tests for new features
- Maintain high test coverage
- Ensure all tests pass before committing
- 新機能に対して包括的な単体テストを書く
- 高いテストカバレッジを維持する
- コミット前にすべてのテストが通ることを確認する

### Documentation / ドキュメンテーション

- Document all public APIs and complex logic
- Keep README.md up to date
- Add inline comments for non-obvious code
- すべての公開APIと複雑なロジックを文書化する
- README.mdを最新の状態に保つ
- 自明でないコードにインラインコメントを追加する

## AI Assistant Guidelines / AIアシスタントガイドライン

### When Using Claude / Claudeを使用する際

1. **Context Awareness / コンテキスト認識**
   - Always review existing code before making changes
   - Understand the project structure and dependencies
   - 変更前に既存のコードを必ずレビューする
   - プロジェクト構造と依存関係を理解する

2. **Incremental Changes / 段階的な変更**
   - Make small, focused changes
   - Test after each change
   - Commit frequently with clear messages
   - 小さく、焦点を絞った変更を行う
   - 各変更後にテストする
   - 明確なメッセージで頻繁にコミットする

3. **Error Handling / エラーハンドリング**
   - Implement proper error handling
   - Provide meaningful error messages
   - Log errors appropriately
   - 適切なエラーハンドリングを実装する
   - 意味のあるエラーメッセージを提供する
   - エラーを適切にログに記録する

4. **Security / セキュリティ**
   - Never commit secrets or credentials
   - Validate and sanitize all inputs
   - Follow security best practices
   - シークレットや認証情報をコミットしない
   - すべての入力を検証およびサニタイズする
   - セキュリティのベストプラクティスに従う

## Communication Style / コミュニケーションスタイル

- Be clear and concise
- Explain reasoning behind decisions
- Ask for clarification when requirements are unclear
- 明確かつ簡潔に
- 決定の背後にある理由を説明する
- 要件が不明確な場合は明確化を求める
